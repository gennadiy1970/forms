form

Этот тег предназначен для:

  1. группировки элементов форм для формирования строки запроса // каждый элемент form формирует собственную строку
  2. хранении в атрибутах информации для формровании строки запроса
  3. передавать информацию о валидности запроса

группировка элементов

  • все теги, находящиеся внутри <form></form> связаны с данной формой
  • теги форм input, находящиеся ВНЕ формы можно свзять с формой с помощью одинакового значения атрибута id формы и атрибута form инпута 
     <form ... id="my_id">
 </form>
<input ... form="my_id">

атрибуты для формровании строки запроса

  • action= адрес запроса // включая протокол
  • method= способ передачи данных
    • GET - значение по умолчанию. Использовать только если данные содержат ASCII символы. Не использовать для важной, конфеденциальной информации
    • POST - данные передаются в теле запроса, а не в самой строке.

Валидность

  • валидация формы происходит по умолчанию
  • novalidate атрибут, отключающий валидацию
  • CSS псевдоклассы :valid и :invalid позволяют визуально оформить валидный и не валидный запрос
