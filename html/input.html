input

Используется для создания интерактивных элементов управления в веб-формах

хранение данных

  • Один из популярных видов хранения данных это запись ключ=значение 
    пример: src="logo.jpg"  // src это key, "logo.jpg" это value
  • Атрибут name тега input содержит ключ
  • Атрибут value тега input содержит значение. Как правило это значение вводит или выбирает пользователь, и поэтому его запись внутри тега не всегда нужна.
  • Ключи в пределах одного документа должны быть уникальны.
  • Значения могут быть любыми, заранее заданными или комбинацией того и другого.

атрибут required

Используется, если без value этого элемента отправка формы не должна осуществляться. Например для полей ввода логина и пароля.

вид и сопособы ввода данных определяются атрибутом type

  • text, password, email, url
  • checkbox и radio
  • color
  • date, time, datetime-local, month и week // частичная поддержка браузерами
  • file
  • number, range
  • tel
