button

Тег button с атрибутами

  • type="submit" формирование строки и отправка запроса
  • type="reset" сброс данных, введенных пользователем

пример

<button type="submit">Send</button>
<button type="reset">Reset</button>

Тег input c атрибутом button

Кнопка без предопределенного поведения

<input type="button" value="Ok">
