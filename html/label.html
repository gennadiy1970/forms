label

Подпись к элементу форм.

Способы связать label с form

  • расположить label с элементом внутри тега form
  • связать атрибут label for с id тега form 
    <form id="myid" ... > ... </form>
<label form="myid"> Текст <input type="text"> </label>

Способы связать элемент с label

  • расположить элемент внутри тега label 
    <label> Текст <input type="text"> </label>
  • использовать атрибут for= id элемента 
    <label for="myid"> Текст  </label>
<input type="text" id="myid">

Способы разместить текст слева и справа от элемента

  • Перестановка текста внутри label
    <label> Текст слева <input type="text" placeholder="Введите текст"> </label>
    <label> <input type="text" placeholder="Введите текст"> Текст справа </label>
  • Перестановка label по отношению к элементу
    • <label for="myid"> Текст  </label>
<input type="text" id="myid" placeholder="Введите текст">
    • <input type="text" id="myid" placeholder="Введите текст">
<label for="myid"> Текст  </label>
