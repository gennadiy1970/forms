Строка HTML запроса

Приложения должны суметь из­влечь данные из строки запроса, поэтому строка имеет определенный синтаксис.

https://aplication.com/?user+name=Homer&password=qwerty%5E
  • https протокол добавит номер порта в запрос // :80 для http и :443 для https
  • ? разделитель: слева адрес получателя данных, а справа пары данных name=value // имя=значение
  • & разделитель пар данных
  • + символ пробела
  • % служеб­ные символы, кириллица и т.п. будут переданы в виде символа процента, за которым следует шестнадцатеричное значение соответствующего символа ASCII

Строка HTML запроса обрабатывается сервером и возвращает результат

Пример: строка покажет курс валют на заданную дату

https://api.privatbank.ua/p24api/exchange_rates?json&date=14.06.2017

Работу по составлению сток браузер делает самостоятельно с помощью элементов форм.
