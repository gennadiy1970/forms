Number

Элемент для ввода числа.

<label> Введите число 
    <input type="number" 
    name="my-number">
</label>

атрибуты

  • max максимальное значение
  • min минимальное значение
  • step шаг
<label> Введите число 
            <input type="number" 
            name="my-number"
            max="3.5"
            min="-3.5"
            step=".25"
            >
        </label>

Range

Изменения вида для ввода числа - вместо поля для ввода используется ползунок

<label> Введите число 
        <input type="range" 
        name="my-range"
        max="3.5"
        min="-3.5"
        step=".25"
        >
    </label>
