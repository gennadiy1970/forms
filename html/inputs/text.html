Text

  • Однострочное текстовое поле.
  • Служит для генерации пар "ключ-значение" со значением в виде текста, внесенного пользователем.
  • Разрывы перевода строки автоматически удаляются из входного значения.
  • атрибут placeholder служит для подсказки пользователю что именно нужно вводить. Исчезает при вводе текста пользователем
  • атрибут value служит значением по умолчанию. Не исчезает при вводе текста пользователем.
  • атрибут maxlength необязательный атрибут. Ограничивает длину строки.

Password

  • Разновидность типа Text.
  • При вводе пароля на экране отображает символ *
  • атрибут maxlength рекомендуется к использованию.
  • атрибут required рекомендуется к использованию.
<label>
                <input  type="password" 
                name="mypass" 
                placeholder="Введите пароль" 
                maxlength="5" 
                minlength="3" 
                required
                > 
                Минимум 3 и максимум 5 символов
                </label>

Email

  • Разновидность типа Text.
  • Для ввода и редактирования email
  • автоматическая валидации адреса // проверка на наличие @
  • атрибут required рекомендуется к использованию.
<label>
     <input  type="email" 
             name="my-email" 
             size="42"
             placeholder="Введите email"
             required
     > 
     Ваш email
 </label>

Url

  • Разновидность типа Text.
  • Введенный url безопасный - по нему нельзя осуществить переход без преобразований строки программой.
  • При вводе URL происходит валидация и замена спецсимволов кодом ASCII
  • атрибут pattern используется для шаблона валидации.
<label>
     <input  type="url" 
             name="myurl" 
             placeholder="Введите url"
             value="http://fecore.net.ua/theory/module-1.html" 
             minlength="40"
             size="42"
     > 
     Url к коспекту теории
 </label>
