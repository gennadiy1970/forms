Radio & Checkbox

  • Требует использования атрибута value для определения значения, которое будет отдано этим элементом
  • Используйте атрибут checked, чтобы указать, должен ли флажок быть выставлен
  • Используйте атрибут disabled, чтобы данный input нельзя было выбрать
  • стилизация стандартным способом невозможна. Существуют трюки с помощью CSS или JS + CSS

radio

выберите один цвет:
<label>red    
    <input type="radio" name="color" value="red"> 
</label>
<label>green  
    <input type="radio" name="color" value="green" checked> 
</label>
<label>blue   
    <input type="radio" name="color" value="blue"> 
</label>
<label class="comment">black  
    <input type="radio" name="color" value="black" disabled> 
</label>

checkbox

Выбирите несколько цветов:
<label>red    
    <input type="checkbox" name="colors" value="red"> 
</label>
<label>green  
    <input type="checkbox" name="colors" value="green" checked> 
</label>
<label>blue   
    <input type="checkbox" name="colors" value="blue"> 
</label>
<label class="comment">black  
    <input type="checkbox" name="colors" value="black" disabled> 
</label>
